# NYC Housing Income Analysis

## Overview
This project analyzes factors associated with household income in New York City using data from the NYC Housing and Vacancy Survey.

## Analysis
I explored relationships between income and housing/demographic factors, including:
- Age
- Maintenance deficiencies
- Year moved to NYC

I developed a multiple linear regression model to evaluate how these factors relate to household income.

## Methods & Tools
- R
- R Markdown
- Exploratory Data Analysis
- Data Visualization
- Linear Regression
- Model Diagnostics
- Variance Inflation Factor (VIF) Analysis
